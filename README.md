# cRCT Push-forward Power Distribution

This repository provides code associated with our paper "A Bayesian hierarchical push-forward method for cluster Randomized Controlled Trial power calculations in complex humanitarian settings"

1.) simulations
  This directory contains all the code needed to recreate Figures 1 and 2. 

2.) R Shiny app
  `app.R` runs the Shiny application, which essential wraps the pipeline from above into an interactive user interface. You will need to download the file and run it locally to use the app.
